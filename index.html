Protocol

  1. Download presell quiz template by clicking download button
  2. Export your zip file and rename download folder to name of your country like ct8, bs9...
  3. In your folder you will find basic html page with our regular architecture:
    • config.js
    • index.html & reg1.html
    • js, css and img folder
  4. config.js is main configuration file where you can setup all parameters about your quiz and reg page:
    • userType: set user type [soi, doi]
    • contentType: set content type [adult, familySafe]
    • apiKey: api key (do not change it)
    • datingApp: default dating app for registration if no flow
    • regRoute: set registration route [/reg1, /register-user-presell, ...]
    • googleBtnText: set custom text for google button
    • redirection: set registration step to be on the same page or redirect to it
    • flow: set to pick dating app automatic or manual
    • cookieBar: show cookie bar with popup in footer
    • storeSteps: storing data in locale storage off/on
    • giftCredits: set how much credits user gets as gift
    • companyData: set custom company
    • questions: pick which questions to show, and reorder them as you need
  5. index.html and reg1.html is where you adding your custom html, but be sure not to change anything which is already there
    (index.html for quiz & registration step, reg1.html is just registration step)
  6. In js, css and img folders you have files where you can add your custom scripts, css and images
  7. Above you have template preview to check template before editing and 3 demo pages after adding custom styling. There you can see what is idea and purpose of this template.
